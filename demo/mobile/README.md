# Google Wallet SDK Demo - Mobile

A mobile demo app for testing the Google Wallet SDK functionality.

## Get started

1. Install dependencies

   ```bash
   pnpm install
   ```

2. Build the app

   You need to build the app before it will work:

   ```bash
   # For iOS
   pnpm ios

   # For Android
   pnpm android
   ```

3. Test the SDK

   Open the **SDK Playground** tab to test out all the SDK functionalities. This is where you can interact with and explore the wallet features.

## Development

- **iOS**: Run `pnpm ios` to build and open in iOS simulator
- **Android**: Run `pnpm android` to build and open in Android emulator

This project uses [Expo Router](https://docs.expo.dev/router/introduction/) for file-based routing.
